Cubloom Site

Files:
- index.html: Marketing/support landing page
- privacy.html: Four-language privacy policy
- assets/cubloom_app_icon.png
- assets/cubloom_splash_logo.png

GitHub Pages:
1. Upload all files while preserving the assets folder.
2. In repository Settings > Pages, publish from the main branch root.
3. Use the generated root URL as Marketing URL and Support URL.
4. Use /privacy.html as the Privacy Policy URL.

Before publishing, verify the privacy wording against the actual SDKs used in the app.
